📱 Platform Exam App

A modern Flutter-based exam platform designed for managing and taking exams in an interactive and organized way.
Built with scalability in mind, supporting both admin and user roles.

🚀 Features
👤 User Features
🔐 Authentication (Email & Google Sign-In)
📝 Take Exams بسهولة
⛔ منع إعادة دخول الامتحان بعد الانتهاء
📊 View Top Scores (Leaderboard)
❤️ Interact with posts (Likes)
👤 Personal Profile
🛠️ Admin Features
➕ Add Exams
❌ Delete Exams
📢 Post updates & announcements
👀 Manage users and exam content

⚙️ Installation
git clone https://github.com/your-username/platform-exam-app.git
cd platform-exam-app
flutter pub get
flutter run
🔐 Environment Setup

Make sure you configure Firebase:

Create a project on Firebase
Add Android/iOS app
Download:
google-services.json
GoogleService-Info.plist
Enable:
Authentication (Email + Google)
Firestore Database
📦 Build Release
flutter build appbundle --release
